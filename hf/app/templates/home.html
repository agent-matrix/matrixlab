HF Backend API for GitHub Repos

This Space can act as a microservice gateway to a MatrixLab Runner at {{ runner_url }}. 

curl -X POST /repo/run \
  -H \"Content-Type: application/json\" \
  -d '{
    \"environment_id\": \"gitpilot-main\",
    \"profile\": \"gitpilot\",
    \"repo_url\": \"https://github.com/ruslanmv/gitpilot\",
    \"branch\": \"main\"
  }'

Profiles available: gitpilot, agent-generator, repoguardian, or custom with your own repo URL.
